# Data Format Specification

## Blood Collection Logs
| Column | Description |
|--------|-------------|
| date | YYYY-MM-DD |
| time | HH:MM |
| patient_count | Number of blood collection patients |

## Outpatient Data
| Column | Description |
|--------|-------------|
| date | YYYY-MM-DD |
| outpatient_total | Total outpatient counts |

## Weather Data
| Column | Description |
|--------|-------------|
| date | YYYY-MM-DD |
| temperature | Celsius |
| precipitation | mm |
